# 📖 더덕 플래너

### 화면 우측에 콕 박아두는 나만의 공부 플래너

**날짜별 계획을 적고, 캘린더로 넘겨보고, 항목을 체크하며**
**항상 화면 맨 위에 띄워두는 데스크톱 위젯 (Electron)**

<br>

📌 항상 위 고정 · 🗓️ 캘린더 · ✅ 체크박스 · 🔗 링크 클릭 · 😀 이모지 입력 · 📝 전역 메모

<br>

따뜻한 감자색 톤의 작고 아늑한 플래너예요. 🥔

</div>

---

## 🚀 빠르게 사용하기

```bash
# 1) 의존성 설치 (최초 1회)
npm install

# 2) 실행
npm start
```

> [!IMPORTANT]
> 처음 실행하면 화면 **우측 상단**에 작은 창으로 떠요.
> 캘린더에서 날짜를 클릭하고 `✏️ 수정`으로 그날 계획을 적어보세요.

> [!TIP]
> 창 가장자리를 드래그하면 **크기 조절**, 상단바 빈 곳을 드래그하면 **위치 이동**이에요.
> 옮긴 위치와 크기는 자동 저장되어 다음에 켤 때 그대로 복원돼요.

요구 환경: **Node.js + npm** (검증: node v25.7.0 / npm 11.10.1, Windows 11) · Electron 33.x 가 자동 설치됩니다.

## ✨ 기능 & 사용법

| 기능 | 사용법 |
|------|--------|
| 📐 **크기/위치 조절** | 가장자리 드래그로 크기, 상단바 드래그로 이동. 위치·크기 자동 저장·복원 |
| 📌 **항상 위 고정** | 상단바 📌 버튼 토글. 켜지면 강조됨. 다른 창 위에 계속 떠 있음 |
| ✏️ **인라인 수정** | `✏️ 수정` → 자유롭게 편집 → `💾 저장` (`취소`로 되돌리기) |
| 🗓️ **캘린더** | 위 달력에서 날짜 클릭 → 그날 플래너 표시. 내용 있는 날은 점(●), 오늘은 테두리 강조 |
| ◀▶ **전날/다음날 이동** | 하단 `◀` `▶` 버튼, 가운데 `오늘` 버튼으로 점프 |
| ✅ **체크박스** | 항목을 체크하면 가로 취소선이 그어지고, 풀면 사라짐 (편집 없이 바로 토글) |
| 🔗 **링크 클릭** | 본문 링크를 누르면 기본 브라우저로 열림 |
| 😀 **이모지 입력** | 편집 모드에서 😀 버튼 → 팔레트에서 이모지 클릭 → 커서 위치에 입력 |
| 📝 **전역 메모** | 플래너 아래 메모칸. **날짜와 무관하게 항상 같은 내용 공유**, 자동 저장 |

### ⌨️ 단축키
| 키 | 동작 |
|----|------|
| `Ctrl + S` | 편집 중 저장 |
| `Esc` | 편집 취소 |
| `←` / `→` | (보기 모드) 전날 / 다음날 이동 |

### 🪟 창 컨트롤 (상단바)
- 📌 항상 위 고정 토글 · — 최소화 · ✕ 닫기
- 상단바 빈 영역 드래그 → 창 이동 (frameless 커스텀 타이틀바)

## 📝 플래너 작성법 (마크다운)

본문은 가벼운 마크다운을 지원해요. 편집 모드에서 이렇게 적으면 —

```
- [ ] ⭐️ [참고 링크](https://example.com)
- [x] 🔥 오늘 할 일 끝내기
    메모: 자유롭게 적어요
```

보기 모드에서는 이렇게 보여요:
- `- [ ]` / `- [x]` → 실제 **체크박스** (체크 시 취소선)
- `[글자](주소)` → 클릭 가능한 **링크** (누르면 브라우저로 열림)

> [!TIP]
> 체크박스는 보기 모드에서 바로 누를 수 있어요. 수정 버튼 안 눌러도 돼요.
> 플래너 아래 **📝 메모**는 날짜와 상관없이 항상 공유되는 자유 메모장이에요.

## 💾 데이터 저장 위치

OneDrive 동기화 충돌을 피하려고 데이터는 Windows `userData` 폴더에 저장돼요.

```
C:\Users\<사용자>\AppData\Roaming\potato-planner\
 ├─ planner-data.json      # 날짜별 플래너 본문 (키 = "YYYY-MM-DD")
 └─ window-settings.json   # 창 위치/크기 + 항상위고정 상태 + 전역 메모
```

> 내부 폴더명이 `potato-planner`인 건 앱 식별자라서예요 (표시 이름만 더덕 플래너).
> **개인 기록(planner-data.json)은 이 폴더에만 있고, 저장소(git)에는 포함되지 않아요.**

## 🗂️ 파일 구성

| 파일 | 역할 |
|------|------|
| `main.js` | Electron 메인 프로세스. 창 생성/복원, IPC, 데이터 저장, 링크 외부 열기 |
| `preload.js` | `contextBridge`로 안전한 IPC API(`window.planner`)만 노출 |
| `store.js` | 로컬 JSON 읽기/쓰기 (원자적 쓰기 + 손상 백업) |
| `index.html` | UI 마크업 (타이틀바 / 캘린더 / 플래너 / 이모지 팔레트 / 메모 / 하단 네비) |
| `styles.css` | 디자인 토큰 + 스타일 |
| `renderer.js` | 렌더러 UI 로직 (캘린더·편집·체크박스·링크·이모지·메모) |

보안: `contextIsolation: true`, `nodeIntegration: false`, CSP 헤더, 링크는 http/https만 외부 열기 허용.

## 📦 exe로 패키징 (선택)

설치 없이 더블클릭으로 쓰고 싶으면:

```bash
npm install --save-dev electron-builder   # 최초 1회
npm run dist                              # dist/ 에 설치파일(.exe) 생성
```

압축 해제형(폴더 통째) 빌드만 원하면 `npm run pack`.

## ⚠️ 알려진 한계
- 데이터는 일반 텍스트 JSON (암호화·클라우드 백업 없음).
- 앱을 켜둔 채 자정을 넘기면 "오늘" 표시가 자동 갱신되지 않음 → `오늘` 버튼이나 재시작으로 갱신.
- 시간대는 PC(Windows) 시스템 시간대를 따름.
- `electron-builder`는 기본 의존성에 미포함 (패키징 시 별도 설치).
